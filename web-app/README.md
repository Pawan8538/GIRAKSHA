# GeoGuard Web Dashboard

Next.js web dashboard for mining authorities to monitor slope stability, manage alerts, and track field workers.

## Tech Stack
- **Framework**: Next.js 14
- **UI**: Tailwind CSS, Shadcn UI
- **Maps**: Leaflet / Mapbox
- **State**: React Context / Redux

## Getting Started

### Prerequisites
- Node.js 18+
- Backend service running on port 4000

### Installation

```bash
cd web-app
npm install
```

### Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

## Key Features
- **Dashboard**: Real-time overview of risk levels and active alerts.
- **Risk Map**: Interactive heatmap showing slope stability.
- **Alerts**: Management interface for system-generated and manual alerts.
- **ML Forecast**: Visualization of predictive models.
